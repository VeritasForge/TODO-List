## 주요 기술 스택 (Tech Stack)

- **언어/프레임워크**: Dart/Flutter

## Coding Convention

- Effect Dart: Style(https://dart.dev/effective-dart/style)를 준수합니다.

## 코드 작성

- 반드시 TDD를 통해서 코드를 작성합니다.
- JIRA 티켓을 통해 요구사항을 파악합니다.
- 요구사항에 대한 RED -> GREEN -> REFACTORING 순으로 작업을 진행합니다.

### 테스트 코드 작성

- BDD 형태로 작성을 합니다. (Given/When/Then)
- expected 데이터들은 가능하면 Given의 마지막에 작성합니다. (이는 Given만 보고도 어떤 결과를 원하는지 확인하기 위함입니다.)
